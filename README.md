# USD/JPY Signal Desk

USD/JPYだけを対象にした、無料ローカル版のFX判断補助ツールです。

## 起動方法

`run_fx_tool.bat` をダブルクリックしてください。

自動で裏側のサーバーを起動し、ブラウザで `http://localhost:8501` を開きます。

## 環境を作り直す場合

`setup_environment.bat` をダブルクリックしてください。必要な無料ライブラリを入れ直します。

## 停止方法

ブラウザを閉じても裏側のサーバーは残ります。止めたい場合は `stop_fx_tool.bat` をダブルクリックしてください。

## スマホ通知

無料の `ntfy` を使ってスマホ通知できます。

1. スマホに `ntfy` アプリを入れます。
2. ツール画面の左側にある `スマホ通知` を開きます。
3. 表示されているトピックをスマホで購読します。
4. `start_notifications.bat` をダブルクリックすると、15分ごとの自動監視が始まります。
5. 止めたい場合は `stop_notifications.bat` をダブルクリックします。

初期設定では、買い/売り優勢かつ信頼度68%以上の時だけ通知します。同じシグナルは180分以内に再通知しません。

## PC不要で動かす場合

GitHub Actions版を使うと、PCを起動していなくても監視できます。iPhoneからはGitHub Pagesの簡易ダッシュボードを開けます。

詳しい設定は `GITHUB_SETUP.md` を見てください。

GitHubへアップロードする前に `make_github_upload_folder.bat` を実行すると、不要なローカル環境を除いたアップロード用フォルダを作れます。

## 表示内容

- 現在価格
- 買い優勢 / 売り優勢 / 様子見
- 候補ゾーン
- 損切り候補
- 利確候補
- 判断根拠
- サポート / レジスタンス
- 相場環境: ADX、ボリンジャーバンド幅、EMA傾き
- 米金利: 米2年債、米10年債、5営業日/20営業日の変化
- 重要イベント: FOMC、日銀会合、手動追加イベント

## 注意

このツールは売買判断の補助です。利益を保証するものではありません。無料データは遅延や欠損があり得ます。実運用前に必ずデモ環境で検証してください。

ntfyの公開サーバーでは、トピック名を知っている人が同じ通知を受け取れる可能性があります。推測されにくいトピック名を使ってください。

## イベントを手動追加する場合

`events_usdjpy.csv` に以下の形式で追加できます。

```csv
date,title,impact
2026-06-05,米雇用統計,high
```
